# Code of Conduct

Outcrop Registry is a public, contributable dataset registry. Contributions should be factual, source-backed, and respectful.

Do not use issues or pull requests for harassment, spam, source-owner impersonation, misleading endorsement claims, or attempts to publish sensitive data.

Project maintainers may edit, close, or remove contributions that are abusive, legally risky, unsupported by sources, or outside the registry scope.
